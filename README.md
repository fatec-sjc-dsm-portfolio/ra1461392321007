# Portfólio Acadêmico - Davi Maciel Ferreira

Portfólio digital desenvolvido com HTML5, CSS3 e JavaScript puro para hospedagem no GitHub Pages.

## Publicação no GitHub Pages

1. Envie os arquivos para o repositório.
2. Acesse `Settings > Pages` no GitHub.
3. Selecione a branch principal e a pasta raiz do projeto.
4. Salve a configuração e aguarde a geração do link de deploy.
